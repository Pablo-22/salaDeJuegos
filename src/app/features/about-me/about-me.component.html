El juego que desarrollé...

El juego que desarrollé es el clásico juego del Pong, uno de los primeros juegos de computadora jamás creados, adaptado a la tecnología Angular, utilizando únicamente elementos div, CSS y Typescript para su implementación.

Este juego simple presenta dos barras y una pelota, y se juega de a dos jugadores. Un jugador obtiene un punto cada vez que el oponente pierde una pelota. Para mover las barras, se utilizan las teclas Q y A para el jugador uno y las teclas O y L para el jugador dos
